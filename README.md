# Node JS Unit Testing Examples

Examples of Unit Testing using nodejs.

## Setup

1. Install [nodejs](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)
1. Install requirements `npm install`

## Run Server

`npm start`

## Run Tests

`npm test`
